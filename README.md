# 근대식 출판 단행본 DB — 프로토타입

근대 신식활자본 단행본의 서지·원문·광고 정보를 열람하는 웹 DB 프로토타입입니다.
표본 자료 4종(『農政新編』·『人工養蠶鑑』·『刑法大全』·『蠶桑實驗說』)의
실제 정리 카드·원문 이미지·강명관 카드(신문 광고)를 담았습니다.

## 실행 방법
`prototype/index.html` 파일을 웹브라우저(크롬·엣지 등)로 그냥 열면 됩니다.
설치·서버 불필요. 인터넷 없이도 동작합니다.

## 구현 기능
- **목록 열람** — 표지 썸네일 그리드
- **검색** — 서명·저자·출판사·주제어·청구기호 통합 검색
- **분류 검색** — 발행연도별 / 저자·역자별 / 출판사별 / 주제별 필터 + 정렬
- **상세 서지정보** — 분류번호·소장처·청구기호·서명·서편자·발행·판권지·형태·문자·가격·목차·서발문·장서인 등
- **원문 열람** — 원문 스캔 이미지 페이지 넘기기 + **원문 입력(교감) 텍스트** 나란히 보기
- **신문 광고** — 강명관 카드(신문명·게재일·정가·발매소)

## 파일 구성
```
prototype/
├─ index.html      앱 본체 (UI + 로직)
├─ data.js         서지 데이터 (4종) — 새 자료는 여기에 추가
├─ assets/<책>/    원문 이미지 (PDF에서 렌더링한 페이지 JPG)
└─ README.md
```

## 자료 추가 방법
1. 새 책 PDF의 페이지를 `assets/<id>/pNN.jpg` 로 렌더링
2. `data.js` 의 `BOOKS` 배열에 항목 추가(기존 항목 형식 참고)
3. 원문 입력(교감)은 `TRANSCRIPTS` 에 추가하고 해당 page의 `tr` 로 연결

※ 원문 입력 텍스트는 스캔면에서 판독한 **표본**이며, 실제 서비스에서는 전 페이지 교감본으로 대체됩니다.
